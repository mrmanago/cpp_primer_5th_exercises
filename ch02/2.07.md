> What values do these literals represent? What type does each have?
>
> (a) "Who goes with F\145rgus?\012"
> (b) 3.14e1L
> (c) 1024f
> (d) 3.14L

(a): Who goes with Fergus?(new line), string

(b): 31.4, long double

(c): ERROR: The suffix f is valid only with floating point literals

(d): 3.14, long double
